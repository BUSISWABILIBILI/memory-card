# Memory Card Game

A React memory card game built as part of The Odin Project React course.

## Live Demo

https://BUSISWABILIBILI.github.io/memory-card

## Features

- Fetches Pokémon data from the PokéAPI
- Displays cards in a random order
- Tracks current score
- Saves best score using localStorage
- Resets score when the same card is clicked twice
- Responsive layout for desktop and mobile

## Built With

- React
- Vite
- JavaScript
- CSS
- PokéAPI
- GitHub Pages

## What I Learned

- Using `useState` to manage game state
- Using `useEffect` to fetch external API data
- Passing props between components
- Creating reusable React components
- Using localStorage to persist data
- Deploying a React app to GitHub Pages
